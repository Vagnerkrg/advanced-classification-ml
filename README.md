# advanced-classification-ml
Advanced Machine Learning classification system using Ensemble methods, XGBoost and semi-supervised learning applied to real-world data problems.
# 📊 Advanced Classification ML

Projeto de Machine Learning focado em **classificação de churn de clientes**, evoluindo de forma prática ao longo de 3 etapas:

- Aula 04 → Ensemble Learning (baseline)
- Aula 05 → XGBoost e otimização de performance
- Aula 06 → Semi-supervisionado

---

## 🎯 Objetivo

Construir um pipeline completo de Machine Learning para prever churn de clientes usando diferentes abordagens:

- Modelos base (Logistic Regression, Decision Tree, Random Forest)
- Modelos avançados (XGBoost)
- Técnicas semi-supervisionadas

---

## 📁 Dataset

- Telco Customer Churn Dataset
- Problema: prever se um cliente vai cancelar o serviço
- Target: `Churn (0 = Não, 1 = Sim)`

---

## ⚙️ Pipeline atual (Aula 4)

✔ Carregamento de dados  
✔ Limpeza inicial  
✔ Encoding de variáveis categóricas  
✔ Separação treino/teste (stratified)  
✔ Treinamento de modelos:
  - Logistic Regression
  - Decision Tree
  - Random Forest  
✔ Avaliação com accuracy e classification report  

---

## 📊 Resultados iniciais

Os modelos foram comparados com base em:

- Accuracy
- Precision / Recall
- F1-score
